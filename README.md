## Getting Started

### Prerequisites

- Python: >= v3.12.4

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/VictorJGB/fourier-transform
   ```
2. Install the required libs
   ```sh
   pip install -r .requirements.txt
   ```
